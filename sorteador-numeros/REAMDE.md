# 🎲 Sorteador de Números

Um projeto simples em *JavaScript* para sortear números aleatórios.  

## 🚀 Funcionalidades
- Gera números dentro de um intervalo definido.  
- Evita repetições de números já sorteados.  
- Interface simples e intuitiva.  

## 🛠️ Tecnologias
- HTML5  
- CSS3  
- JavaScript
